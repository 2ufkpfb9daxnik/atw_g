## すぐやる

- e-typingのように、textとkana, そしてpreeditの進行度に関する一致を取る
- タイマー中に文字入力を受け付け
- 改行した状態で常に次の内容までだけが見える、それより先は別に見えなくて良い
- 文字入力を受け付けて全部完了したら次のお題を続けて表示する(ランダムに選ぶ)。間にはスペース(表記上は_: アンダーバー)を入れる
- 計測中はesc以外すべて文字入力に吸われる
- 流れる文字モード
- タイマーが終了したら文字入力受付も終了する
- 文字数をカウントしてスコアとして表示する。これは右か、タイマーの上に表示する
- 次に打つ文字列を表示する
- 確定したら薄文字、打ってないのは普通の色、間違ってたら最後の入力まで全部赤
- 記録(sqlite?普通にテキスト?csv?)
